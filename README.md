# `stdcolt`
`stdcolt` is a performant `C++` library to extend and complement the `C++` standard library.

It contains cross-platform utilities for:
- memory allocation,
- coroutines,
- contracts,
- simd,
- type erasure,
and more.

# Extensions:
`stdcolt` comes with extensions that are more specific and opinionated, these can be found in
`src/extensions`.

Extensions are always in the namespace `stdcolt::ext`, or for `C` APIs prefixed with `stdcolt_ext_`.
